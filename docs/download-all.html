Reference Data for NGS

To run some NGS data analysis (taxonomy classification or ChIP-Seq) it is required to provide reference data (e.g. RefSeq databases, NCBI taxonomy files, etc.).
It is recommended to use the Online Installer to download these data, however, one can also use the download links below and manually configure the packages. Make sure there is enough disk space on the target computer.

Data for NGS metagenomics classification

  • NCBI taxonomy classification
  • NCBI RefSeq bacterial genomes
  • NCBI RefSeq viral genomes
  • NCBI RefSeq GRCh38 human genome
  • Kraken DB: MiniKraken 4Gb database
  • CLARK-l DB: RefSeq bacterial+viral genomes
  • CLARK-l DB: RefSeq viral genomes
  • DIAMOND DB: UniRef50
  • DIAMOND DB: UniRef90
  • MetaPhlAn2 embedded DB: mpa_v20_m200

Data for ChIP-Seq analysis

  • Human data for Cistrome workflow

Source code

UGENE is an open-source software with GNU GPL v2 license. The source code is available on GitHub.

Old UGENE Packages

Before UGENE version 34 the following packages were provided:
  • Full Package: UGENE main executables, external tools (including tools for NGS data analysis).
  • NGS Package: the same as in the Full Package + ChIP-Seq data analysis additions (RScript, reference data)
Since version 34 only one online and one offline package per 64-bit operating system was left (see above).
The old packages of UGENE version 33 are available below. There are also earlier packages for obsolete versions of the operating systems.

Windows

Besides types specified above, the following package delivery options were available in UGENE v.33 (note that the delivery type does not affect a package content):
