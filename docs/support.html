Free Support

How can I ask a question?
You've started to use UGENE and have a question about it. Have you tried to find the answer in the documentation? If you still don't know the answer, then or create a post on our forum.
I think this is a defect. Can you fix it?
about the defect, create a post on the forum or create an issue directly in our bug-tracking system (you would have to register). Critical issues are fixed as soon as possible. After an issue has been fixed, UGENE snapshot version with the issue fixed can be downloaded here. Other non-critical issues are scheduled on one of the future releases. If you want a defect to be fixed as soon as possible you can use the commercial support service.
I want a new feature
The feature could be a minor improvement or a new plugin. about your ideas or create a post on the forum. You can also create an issue directly in the bug-tracking system. Remember that the more users ask for a feature to be implemented, the more likely it would be included into the next release. If you don't want to wait, use the corresponding commercial support service.

Commercial Support

I think this is a defect. Fix it ASAP!
If you don't want to wait and want a defect to be fixed as soon as possible then this paid service is for you.
I want a new feature ASAP!
There are a lot of good features that we would like to include in UGENE, but sometimes we don't have enough resources to implement all the features as soon as you would like them to have. So if you want a feature to be implemented as soon as possible you can use this paid service. The new feature can be either a minor improvement or a big change. Depending on the requirements the feature can be either included into the common UGENE package, so everyone would be able to use it, or it can be customization available only for you.
I want to automate my computational tasks!
Do you have to do… to do… to do… to do so much manual bioinformatics tasks that can be automated? Our consultants would analyze your workflow and would help to automate it.
I want to use cloud resources to calculate my tasks!
If you don't have enough local resources to run resource-consuming tasks we can help you to use a cloud (e.g. Amazon EC2 or Windows Azure) to get the results. We will develop a solution to run your tasks directly from the UGENE Workflow Designer.
I want to create a new (project / custom plugin / etc.)
Do you want some global changes? Do you dream of some project similar to UGENE or would like to use UGENE in other project? Do you want to use the UGENE team experience to make your dream come true?
Write to UGENE for further information.
