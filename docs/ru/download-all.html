Внешние инструменты

UGENE предоставляет графический интерфейс для множества биоинформационных инструментов. В зависимости от требуемого анализа данных и установленного пакета UGENE, может потребоваться дополнительная установка/конфигурация соответствующих инструментов:
  • Пакет внешних инструментов: пакет содержит внешние («external») инструменты, за исключением JRE и RScript. Данный пакет по умолчанию включен в онлайн-инсталлятор, а также полный и NGS пакеты. Скачанный пакет инструментов необходимо указать в настройках приложения.
  • Java Runtime Environment (JRE): хотя UGENE не является Java-приложением, несколько для нескольких интегрированных инструментов (например, FastQC, Trimmomatic) требуется установленная версия JRE 8 или выше.
  • RScript: R используется в UGENE инструментами для анализа ChIP-Seq данных.

Старые пакеты UGENE

До версии UGENE 34 выпускались следующие типы пакетов:
  • Стандартный: содержит базовые модули UGENE, встроенные в саму программу («embedded») инструменты, примеры входных данных и др.
  • Полный: в дополнение к модулям из стандартного пакета содержит внешние («external») инструменты (включая большинство инструментов для анализа данных высокопроизводительного секвенирования (NGS). Рекомендуется использовать данный пакет в случае отсутствия на компьютере доступа к Интернету.
  • NGS: предназначен для анализа ChIP-Seq данных.
Начиная с версии 34 поставляются по одному онлайн и одному офлайн пакету для 64-битных версий операционных систем (см. выше).
Старые пакеты версии 33 доступны для скачивания ниже. Также доступны пакеты других версий для устаревших операционных систем.
