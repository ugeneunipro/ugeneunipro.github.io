Старые версии UGENE

До версии UGENE 34 выпускались следующие типы пакетов:
  • Полный: содержит базовые модули UGENE и внешние («external») инструменты (включая большинство инструментов для анализа данных высокопроизводительного секвенирования (NGS).
  • NGS: предназначен для анализа ChIP-Seq данных.
Начиная с версии 34 поставляются по одному онлайн и одному офлайн пакету для 64-битных версий операционных систем (см. выше).
Старые пакеты версии 33 доступны для скачивания ниже. Также доступны пакеты других версий для устаревших операционных систем.

Windows

Помимо типов пакетов, указанных выше, для Windows поддерживается два способа поставки, не влияющих, однако, на содержащиеся модули:
  • Оффлайн-инсталлятор: предоставляет более полную интеграцию с операционной системой.
  • Zip-архив: портативная версия программы, не требующая администраторских прав.

Референсные данные для NGS

Для некоторых видов анализа данных высокопроизводительного секвенирования (таксономической классификации и ChIP-Seq) требуется наличие референсных данных (например, RefSeq-данные, таксономическая информация с NCBI и др.).
Рекомендуется использовать онлайн-инсталлятор для скачивания этих данных. Однако, можно также использовать ссылки для скачивания ниже. При этом потребуется дополнительная конфигурация установленного пакета. Убедитесь, что на компьютере достаточно дискового пространства.
